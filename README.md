# Poke Logger (OCR)
This is a tool that purely exists to take data from a stream and dump that data into an easily usable CSV file. 
This can then be imported into g-sheets or excel for easy use. 

## Known Bugs:
- Moon and nest ball getting confused
- General ball detection issues due to Arceus style duplicates and some similar color histograms
